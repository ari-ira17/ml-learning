# Машинное обучение и большие данные (Machine Learning & Big Data)

Репозиторий содержит комплекс лабораторных работ, выполненных в рамках учебного курса. Проекты охватывают весь путь разработки моделей: от первичного разведочного анализа данных (EDA) до построения глубоких сверточных нейросетей для компьютерного зрения.

---

## Технологический стек
* **Язык программирования:** Python 3.9+
* **Анализ данных и классическое ML:** Scikit-Learn, NumPy, Pandas, Matplotlib, Seaborn
* **Глубокое обучение (Deep Learning):** TensorFlow, Keras
* **Среда разработки:** Jupyter Notebook 

---

## Структура репозитория и содержание лабораторных

### Часть 1. Классическое машинное обучение (Scikit-Learn)

* **Лабораторная работа 1: Введение в исследовательский анализ данных (EDA)**
    * Первичный анализ датасета, обработка пропусков, поиск аномалий, визуализация распределений и корреляционных матриц.
* **Лабораторная работа 2: Обучение с учителем. Задача регрессии**
    * Предсказание непрерывной величины. Работа с линейными моделями, регуляризацией (Lasso/Ridge) и метриками качества ($R^2$, MSE, MAE).
* **Лабораторная работа 3: Обучение с учителем. Задача классификации**
    * Бинарная и многоклассовая классификация. Метрики Precision, Recall, F1-score, ROC-AUC. Балансировка классов.
* **Лабораторная работа 4: Решающие деревья и ансамбли моделей**
    * Построение одиночных решающих деревьев, случайного леса (Random Forest) и градиентного бустинга. Настройка гиперпараметров.
* **Лабораторная работа 5: Обучение без учителя. Задача кластеризации**
    * Группировка данных без разметки с помощью алгоритмов K-Means, DBScan, иерархической кластеризации. Оценка качества силуэтом.
* **Лабораторная работа 6: Понижение размерности и отбор признаков**
    * Борьба с «проклятием размерности». Применение метода главных компонент (PCA), извлечение и фильтрация наиболее информативных признаков (Feature Selection).

---

### Часть 2. Глубокое обучение и нейросети (TensorFlow / Keras)

* **Лабораторная работа 7: Полносвязные нейронные сети (FCNN)**
    * Проектирование многослойных перцептронов для решения задач регрессии и классификации. Настройка функций активации (ReLU, Softmax), оптимизаторов и функций потерь.
* **Лабораторная работа 8: Сверточные нейронные сети (CNN) и Компьютерное зрение**
    * Разработка архитектур для классификации изображений. Работа со слоями `Conv2D` и `MaxPooling2D`. Использование подходов Transfer Learning (перенос знаний) и Fine-Tuning на базе предобученных топ-архитектур (VGG19, InceptionV3) для распознавания объектов.
